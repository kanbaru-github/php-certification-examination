# PHP8技術者認定初級試験 実習環境

PHP8初級試験の実習環境です。  

## 環境

「独習PHP 第4版」より動作確認環境を参考に選出しました。

* PHP 8.0.3
* Apache 2.4.46
* MariaDB 10.4.18

特に試験に重要なものは、PHP バージョンです。

## 実行環境

実行環境の実際の構築方法は、各サブディレクトリ内に使用方法を `README.md` として記載してあります。

### Docker

Docker コンテナを利用して構築する方法です。  
複数コンテナを使用するため、Docker Compose も利用します。  
Windows / Mac とも Docker Desktop が必要です。
